# ✨ Hifza - Digital Marketing Expert
**[Landing](https://hifza-website.netlify.app/)**
Hifza is a high-performance Landing page made with Astro and Tailwind for a digital marketing expert.

## Features
- Clean and simple design.
- Fully Responsive.
- Fast and efficient thanks to Astro.
- Customizable with Tailwind.
- Highly optimized SEO.

## 🧞 Commands

To run locally:

| Comando          | Acción                                       |
| :--------------- | :------------------------------------------- |
| `git clone repo` | Clone Repo                          |
| `cd repo`        | Access root dir                    |
| `npm install`    | Install Deps                        |
| `npm run dev`    | Run locally in `localhost:4321` |
